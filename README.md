# 🌐 Nextra 기반 문서 사이트

이 프로젝트는 **[Nextra](https://nextra.site)** 프레임워크를 기반으로 제작된 **정적 문서 사이트**입니다.  
Next.js 위에서 동작하며, 손쉽게 콘텐츠 중심의 기술 문서나 개발 블로그를 만들 수 있습니다.

---

## 🚀 소개

Nextra는 **Next.js**를 기반으로 한 문서 전용 프레임워크로,  
마크다운(`.md`, `.mdx`) 파일을 이용해 손쉽게 페이지를 구성할 수 있습니다.  
자동으로 생성되는 탐색 메뉴, 깔끔한 다크 모드, 코드 하이라이팅 등  
개발자 친화적인 UI를 제공합니다.

이 사이트는 프론트엔드 및 개발 관련 내용을 보기 쉽고 구조적으로 정리한  
**개발 문서 허브**로 설계되었습니다.
