# introduction

안녕하세요! **웹스토리보이**입니다.  
이번에는 현대적인 React 기반 프레임워크인 Next.js에 대해 단계적으로 공부해보겠습니다.

Next.js는 **서버 사이드 렌더링(SSR)**, **정적 사이트 생성(SSG)**, 그리고 클라이언트 사이드 렌더링(CSR)을 모두 지원하는 강력한 풀스택 프레임워크로, 대규모 웹 서비스부터 개인 블로그까지 폭넓게 활용되고 있습니다.

이 시리즈에서는 Next.js 공식 문서를 기반으로, 핵심 개념부터 실전 예제까지 하나씩 직접 다뤄보며  
> "왜 이런 구조로 되어 있는지"  
> "실제 프로젝트에서는 어떻게 적용되는지"  

를 함께 살펴보겠습니다.

복잡한 개념도 이해하기 쉽게 풀어내고, 실제 개발에 바로 적용할 수 있도록 명확한 코드 예시와 함께 정리할 예정입니다.  
그럼 지금부터 Next.js의 세계로 한 걸음씩 들어가 보겠습니다. 🚀


## 시작하기 전에 알아두면 좋은 기본 지식

이 문서는 여러분이 어느 정도 **웹 개발에 대한 기본 이해**가 있다고 가정합니다.  
Next.js를 시작하기 전에 아래의 기술에 익숙하다면 도움이 됩니다:

- **HTML** – 웹 페이지의 구조를 정의하는 마크업 언어  
- **CSS** – 웹 페이지의 스타일을 지정하는 스타일시트 언어  
- **JavaScript** – 웹의 동적 기능을 구현하는 프로그래밍 언어  
- **React** – 컴포넌트 기반으로 UI를 구축하는 JavaScript 라이브러리  


## 🎓 React가 처음이라면?

React에 익숙하지 않거나 복습이 필요하다면,  
저희가 준비한 **[React Foundations](https://react.dev/learn)** 강좌와 **[Next.js Foundations](https://nextjs.org/learn/foundations/about-nextjs)** 강좌를 추천드립니다.  
학습을 진행하면서 직접 애플리케이션을 만들어 볼 수 있어, 기본기를 탄탄히 다지는 데 큰 도움이 될 것입니다.
